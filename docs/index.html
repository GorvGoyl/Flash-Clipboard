Features

  • Free and Open Source Software
  • Saves history of your last copied texts even after reboot
  • Opens on the right side of the mouse cursor which makes it super efficient to paste text
  • 100% privacy of data. Your copied texts are stored locally and it doesn't send any data through the internet
  • Pure text pasting without any styles/formats
  • Supports Emojis 😎
  • Set your favorite shortcut to open Flash Clipboard
  • Auto run on startup so you don't have to run it everytime
  • Stores up to last 200 items
  • Pause option to stop capturing text for a while
  • Auto update itself
  • More exciting features are under development like search, pin texts, password store etc
